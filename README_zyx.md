# Sub2API 本地构建与测试指南

## 1. 环境要求

建议先准备以下环境：

- Go 1.26.5
- Node.js 20+
- pnpm 9+
- Docker Desktop

说明：

- 后端版本以 [backend/go.mod](backend/go.mod) 为准，当前是 `go 1.26.5`
- 本地构建时请优先以 `backend/go.mod` 为准
